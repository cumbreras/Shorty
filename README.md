# Shorty
Shorty is a URL Shortener Service written in Go

## Executing it
Use the go compile to compile the service, you could leverage the Makefile for this:

```go
make build
```

## Usage

### Standard-like
```go
make start
```

### Dev-Autoreload-like

```go
make start-dev
```

## Testing
```go
make test
```
